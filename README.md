## Steps to control the drone using Scratch
Windows:

1. connect to Tello via Wi-Fi
2. windows + R
3. cmd
4. cd THIS_FOLDER
5. node Tello.js
6. launch Scratch
7. Shift + File->Import experimental HTTP extension
8. choose Tello.s2e
9. File -> Open -> keyboardcontrol
10. press "t" for takeoff
